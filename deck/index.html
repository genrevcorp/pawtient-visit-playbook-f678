模組一:必勝心態

勢在必得,但贏的定義要對

📝 先談心態,因為心態錯了,後面所有技巧都會用歪。核心矛盾:我們極度想要這單,但越想要越要站得住、走得開。
2 / 66
必勝心態

贏在『讓對方願意給你下一步』

不是贏在這一場說服

  • 成功定義=拿到試點門票+關係推進一格,不是當場拿承諾
  • 把單場輸贏押滿會逼自己過度承諾、自爆
  • 必勝=讓國泰覺得『不跟你合作才有損失』
  • 每場會只設一個最小可達成的下一步
📝 訓練重點:會議結尾一定主動收斂『那我們下一步是不是先 X?』把 close 縮到對方能當場點頭的尺寸。絕不在桌上逼簽。讓大家練習把『下一步』講出口。
3 / 66
必勝心態

用『零風險可逆』當底氣,不是用『畫大餅』

氣勢押在結構,不押在 AI 多神

  • 必勝牌=3個月免費、可逆、不碰核心系統、資料屬保戶、不作核保依據
  • 對方一畫餅,守門人就把你打回原形
  • 對方猶豫時回到結構:可停、不綁約、資料是保戶的
  • 願景(健保卡/健康存摺)只當上行選擇權帶過
📝 反覆操練:每次對方猶豫,不要訴諸情懷或願景,要回到『你損失什麼?幾乎沒有』這個結構性事實。願景絕不寫進承諾。

必勝心態

誠實到讓對方驗得到 — 因為對方一定會驗

弱點先講,講成合作理由

  • 精算/法遵/資安會驗穿任何吹的數字,一穿全盤失信
  • 弱點藏不住:零營收、零LOI、用戶1000、團隊缺角
  • 把弱點講成『正因如此才需要試點驗證』
  • 會被驗證的(準確率/留存/賠付率)一律不報單一數字
📝 這是必敗紅線的正面版本。主動點名自己缺保險/獸醫/精算/資安,並說明試點期如何補位,讓誠實變成可信度資產。寧可少說也不誇大。

絕不讓對方嗅到『非你不可』

合作是加速,不是續命

  • 對方一旦感覺我們缺氧、急著被救,籌碼瞬間歸零

心理與影響力技巧

互惠 + 好感

先給禮物、建立同盟感

  • 互惠:試點當『你先給的禮物』— 出工出迭代、零成本、隨時喊停
  • 讓對方感覺『已欠你一個認真評估』,而非你在求他試
  • 好感:先攻 Champion 建同盟『一起把這件事做成』
  • 對核保/法遵:『我懂你在怕什麼,我先幫你想好了』
📝 互惠界線:禮物要真無條件,送禮當下不可口頭討債。回報請求延後到試點出結果後,且只能要『書面評估/下一步』。好感界線:讚美要具體真實,不越級攀關係、不踩核保地盤。

承諾與一致 + 登門檻

小門先過,大事自然跟上

  • 不要簽大約(大門),只要一張試點門票(小門)
  • 小承諾逐步書面化:定指標 → 設門檻 → 達標談付費
  • 讓轉付費成為『他們說過要做的事』,不是你逼的
📝 界線:承諾要對方真心自願,逼當場點頭會被識破。門檻必須對國泰公平,不能設成自己穩贏。黑名單:不可謊稱有 LOI 偽造『別人都已承諾』的一致壓力。

社會認同 + 權威

用可驗證的事實,不灌水

  • 社會認同:政府背書 + Cornell 來源標準 + 真實飼主在用 + CKD 全行業痛點
  • 權威=資料溯源權威:每筆血檢並排原始單據、標 Cornell 參考區間
  • 『讓你們的獸醫顧問自己驗』
  • 可信度開場一句帶過,不堆團隊履歷
📝 界線:不灌水用戶數、不謊稱付費客戶、不把 Cornell/政府包裝成核保背書、不報準確率、不假裝團隊有保險精算專業。權威建在『可被第三方查證的機制』上,不是話術。

稀缺

真稀缺=迭代速度+先發位置

  • 真稀缺:週級迭代能量、已握真實髒資料、CKD 剛性需求窗口
  • 框成『早期合作方能直接影響產品方向』
  • 用真實 BATNA 製造健康稀缺(合作是加速不是續命)
  • 限時可用『這季試點名額/工程帶寬有限』
📝 界線:絕不假造急迫、不謊稱有別家搶。零 LOI 的底牌一旦被戳破,稀缺瞬間變心虛。更不能讓對方看穿你『非常需要這單續命』。

錨定 + 對比效應

先框對戰場、再讓 ask 顯得微不足道

  • 錨定:先丟『3個月免費可逆不碰系統』把門檻錨到地板
  • 用『既往症排除/90天等待期爭議』當價值錨,守住你最強戰場
  • 對比:把試點擺在『傳統系統整合大案(簽年約/串系統/過資安)』旁邊
  • 真正 ask 縮到『15–30分鐘/一張門票』,顯得近乎不好意思拒絕
📝 界線:錨點要合理可信,別丟荒謬數字(631億當TAM=黑名單,被精算一眼識破)。對比參照要真實存在,不能虛構假的昂貴替代方案。

損失趨避 + 框架

把現狀框成『正在流血的損失』

  • 對國泰:不做的話既往症/等待期爭議持續造成糾紛、客訴、品牌耗損
  • 試點=止血,不是額外開銷
  • 對試點:可逆不碰系統,唯一損失是『不試的機會成本』
  • 對 Champion:早一步推動的功勞 vs 被同業先做的遺憾
📝 界線:不用恐懼操弄或誇大威脅(『不做就被淘汰』式FUD)。絕不承諾能降賠付率/損率反向暗示『不用就虧錢』=黑名單。框損失要基於真實已存在的痛點。

人性與面子

地盤是隱形地雷

你正好踩在核保/商品線的核心地盤上

  • 你解的問題(既往症佐證)踩在核保安身立命的專業領地
  • 你越說『我能幫你判斷既往症』,核保越覺得你在說『你們做得不好』
  • 守住『中立第三方資料整理者』,判斷權 100% 留核保手上
  • 把核保塑造成『用你工具後判斷更有依據』的受益者與主導者
📝 DONT:絕不說/暗示『AI 能幫你決定核保/判定既往症/降賠付率』— 同時踩地盤又把責任攬到自己身上。不用『取代/自動判斷/更準』這類字眼。

人性與面子

沒人想當白老鼠,但都想當『聰明的先行者』

消滅下行風險,不吹領先美夢

  • 金控:跟隨同業安全、領先同業危險(出事你一人扛)
  • 拖延常不是不認同,是『憑什麼我們先試、出事誰負責』
  • 把 ask 包成『風險趨近於零的試水溫』
  • 給承辦『聰明又安全』的人設:零成本零風險,還比同業早摸到
📝 DONT:絕不用『你們會是業界第一/搶得先機』去推保守的守門人 — 放大『領先=曝險』只會讓他更怕扛責。也別把試點講得太大、太像動真格。

人性與面子

面子與階級:說服發生在私下

當眾被辯贏的主管會記恨並讓你出局

  • 多人會議裡沒人會當眾承認自己不懂、被你說服
  • 就算你 100% 正確,當眾教育主管=製造敵人
  • 把對方塑造成『提出好問題的人』『有遠見的人』
  • 需要對方改變想法的事,留到一對一/會前私下溝通
📝 被質疑時接法:『這個問題問得非常關鍵,正好是我們設計這個可逆試點時最在意的。』把挑戰轉成對方的功勞。決策者進會議室時結論已寫好 — 真正角力在走廊、午餐、一對一。
